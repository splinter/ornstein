> A command line tool for invoking DHC collections


## Install

```sh
$ npm install --save ornstein
```


## Usage

```js
var ornstein = require('ornstein');

ornstein('Rainbow');
```

```sh
$ npm install --global ornstein
$ ornstein --help
```
